FITT ERP Project Guide

1. Login to the ERP System

  1. Open the web browser and enter the ERP system URL: https://fitt-iitd.in/ERP/login
  2. Choose “IITD/FITT” from the dropdown menu at the user type selection.
  3. Enter your credentials (username and password).
  4. Click on the "Sign In" button to access the system dashboard.

2. Initial Dashboard Overview

  • Notifications Panel: Alerts and updates about completed, pending, or rejected tasks.
  • Main Dashboard: Provides a summary of your ongoing projects, highlighting key metrics and status updates.
  • Side Navigation Bar:
    • Dashboard: Access this button to return to the main dashboard anytime. Think of it as a home button.
    • Projects: This is the primary section you will use. Click here to manage and view details related to your projects.

3. Overview of Project Cycle

4. Fill Pre-Proposal Form

This is the starting point where the faculty submits the pre-proposal form.

  • If rejected, the process ends here. Faculty needs to fill out the form again.
  • If approved, the process moves to the next stage.

5. Navigate to the Project Overview

After approval, the user navigates to the project overview section within the system. (Project → All Projects → View your required project)

6. Project Overview Stages

In the Project Overview, you will see four stages:

  • Assigned
    • The PI has to choose the CO-PI, if any. If there’s no CO-PI, then click on the “View” (Grey Button in the Details Column). Upon clicking on it, you will see a window with all the initial details and a “Move to next Stage” button at the bottom. Click on that button to move ahead to the proposal Stage.
  • Proposal
    • Go to Action and fill out the three forms individually as given.
    • First, Add all the Milestones (Payment Terms).
    • Then Fill Up Part 1 (Note: You won’t be able to submit or save Part 1 unless the milestones aren’t filled).
    • Fill out Part 2: Costing Form. (Note: Remember to Type the Project Cost in the “Total Estimate” field of the Form before filing the values in Expense heads).

Once the Proposal Stage is done, wait for the approvals of the Project Officer and COO.

7. MoU Stage

Submit the MoU/Agreement/GLA to the FITT office for the next stage. The FITT will do the MoU stage. Once Part 2 is approved, you may add the Invoice request to the payment schedule in the MoU stage.
